# FocusFlow - Smart Study Planner

## Project Overview

FocusFlow - Smart Study Planner is a Windows Forms desktop application developed using C# and .NET Framework. The application helps students organize study tasks, manage deadlines, track progress, and improve productivity through a built-in Pomodoro study timer.

## Objectives

* Help students manage daily study tasks efficiently.
* Improve productivity using time management techniques.
* Track completed and pending tasks.
* Provide a simple and user-friendly interface.

## Features

### Login System

* Secure login screen for users.
* Username and password validation.

### Task Management

* Add new study tasks.
* Update existing tasks.
* Delete tasks.
* Mark tasks as completed.

### Search Functionality

* Search tasks quickly using the search box.

### Task Statistics

* Display total tasks.
* Display completed tasks.
* Display pending tasks.

### Date Display

* Automatically shows the current date.

### Pomodoro Study Timer

* Start Timer
* Stop Timer
* Reset Timer
* Study session completion notification

### About Section

* Displays project information.

## Technologies Used

* C#
* .NET Framework
* Windows Forms
* Visual Studio
